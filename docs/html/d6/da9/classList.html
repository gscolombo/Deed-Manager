Detailed Description

template<typename T>
class List< T >

A class representing a List of items displayed in a ncurses window.

This class is a template class that represents a list of items to be displayed in a ncurses window. It takes a list of items of type T and displays them in the window. It takes an optional display function that converts the items to strings for display. If no display function is provided, it uses the default conversion to string of the item.
