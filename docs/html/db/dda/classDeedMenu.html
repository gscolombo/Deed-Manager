Detailed Description

Menu class for deed operations.

This class provides a menu for performing deed related operations. It uses the IDeedOperations interface to interact with the services layer. The supported operations are:

  • Create a new deed.
  • List and read all deeds.
  • Update a deed.
  • Delete a deed.
