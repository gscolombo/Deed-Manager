Detailed Description

Singleton class that holds the terminal UI.

This class is a singleton that holds the terminal UI. It is responsible for initializing and closing the terminal UI. It also has a static method to get the instance of the UI singleton.

All the public and protected members of this class are static and should be accessed through the static getUI() method.
