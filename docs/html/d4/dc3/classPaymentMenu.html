Detailed Description

Menu class for payment operations.

This class provides a menu for performing payment related operations. It uses the IDeedOperations and IPaymentOperations interfaces to interact with the services layer. The supported operations are:

  • Create a new payment.
  • List and read all payments of a deed.
  • Update a payment.
  • Delete a payment.
